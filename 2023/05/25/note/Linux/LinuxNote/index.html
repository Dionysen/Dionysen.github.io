    Linux 文件权限

    常见的有 644、755、777

    444 r–r–r–
    600 rw——-
    644 rw-r–r–
    666 rw-rw-rw-
    700 rwx——
    744 rwxr–r–
    755 rwxr-xr-x
    777 rwxrwxrwx

    解释：

    数字 权限 字母
    4 读 r
    2 写 w
    1 执行 x
    0 无 无
    以 755 为例
    权限代码 7 5
    ———— ————– ———-
    权限对应用户 文件所有者 组用户
    计算 4+2+1 4+1
    权限 可读可写可执行 可读可执行
    若用 chmod 4755 filename 可使此程序具有 root 的权限
