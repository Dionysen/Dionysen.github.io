    a 动画
    b 漫画
    c 游戏
    d 文学
    e 原创
    f 来自网络
    g 其他
    h 影视

    l 抖机灵

    请根据自己的需要自行选择要显示的内容：

    返回参数名称 描述
    id 一言标识
    hitokoto 一言正文。编码方式 unicode。使用 utf-8。
    type 类型。请参考第三节参数的表格
    from 一言的出处
    from_who 一言的作者
    creator 添加者
    creator_uid 添加者用户标识
    reviewer 审核员标识
    uuid 一言唯一标识；可以链接到 uuid 查看这个一言的完整信息
    commit_from 提交方式
    created_at 添加时间
    length 句子长度
