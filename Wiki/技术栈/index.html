技术栈

Author: Dionysen Date: November 13, 2022  0:00:00

学习任何技术都需考虑如下三点：

  1. 能干什么
  2. 在什么平台
  3. 怎么用

如：

Name Usage Platform Using
编程语言 编写程序，执行，让计算机做事 任何操作系统 Code->Compile->Debug->Run
gcc 编译器，编译代码为(c,c++)可执行程序 Unix,Windows,Linux,macOS gcc [filename] [arguments]
git 版本控制系统 Unix,Windows,Linux,macOS add,commit,push,pull
nfs linux与linux或linux与windows之间共享文件 Linux 安装nfs和rpcbind，配置，开放端口，另一端挂载
samba windows指定的文件共享协议，可以在windows与别的平台共享文件 Windows,Linux 安装samba服务，配置，连接
C++ 编程语言，编写成语，在特定平台编译后可以执行 All g++ main.cpp
cmake 一个自动生成makefile的工具，帮助c++项目的快速制定编译选项 All cmake [argument]
makefile 是一种脚本语言，可以打包执行复杂的编译命令 Linux 编辑Makefile文件，使用make命令执行文件的命令
Data structure 是相互存在一种或多种特定关系的数据元素的集合，是计算机储存和组织数据的方式 All 在编写的程序中使用它们
Algorithm 是程序解决问题的方法的准确描述 All 在编写程序时使用的它们
Java 一门面向对象的编程语言 All 编写程序，编译运行
Python 一门解释型编程语言 All 编写脚本，直接由解释器解释执行
