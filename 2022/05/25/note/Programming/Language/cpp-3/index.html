    一、模板

    1.1 模板的概念

    模板就是建立通用的模具，大大提高复用性
    模板的特点：

    • 模板不可以直接使用，它只是一个框架
    • 模板的通用并不是万能的

    1.2 函数模板

    • cpp提供两种模板机制:函数模板和类模板
